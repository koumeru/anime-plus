anime-plusは、自己学習用として作成するアニメ評価サイトです。

# 開発背景
自己学習として、動画教材やYoutubeを参考に様々なDemoアプリを作成してきましたが、内容をトレースしているだけではあまり理解できている気がせず、調べて学びながら、オリジナルなアプリケーションを作ろうと思いました。題材は、趣味であるアニメとサッカーで迷いましたが、前者としました。

# 実装予定機能
- ユーザログイン・ログアウト
- ページネーション(1ページに10件までを表示)
- 検索機能(ユーザが作品を検索できる機能)
- 投稿機能(ユーザが作品の評価を投稿できる機能)
- 良いね機能(任意のユーザ投稿に対して、いいねをできる機能)
- ランキング機能(平均評価の高い作品をランキング形式で表示)

# サイト構成
- トップページ
- 作品ページ(投稿)
- 作品ページ(概要)
- 作品ページ(詳細)
- ランキングページ

# ブランチ戦略
小・中規模な開発になると思われるので、ブランチ戦略は以下とします。
1. **main**：本環境（※基本触らない、デプロイ用）
2. **release(複数)**：バージョンアップ、バージョン管理用(※開発中は基本触らない)
2. **develop**：開発中の最新
3. **feature(複数)**：developから分岐。機能別ブランチ
5. **hotfix(複数)**：developから分岐。修正用ブランチ
